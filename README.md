# 3Dprinting

## FDM打印(Fused Deposition Modeling, 熔融沈積成型)
**原理**：加熱膠絲(filament)，一層層疊起 簡單講就係：用膠絲加熱融化，一層一層「擠」出嚟嘅 3D 打印方式。
FDM（熔融沉積成型）它是目前最常見、最平價、最適合新手入門的一種 3D 列印技術。它透過加熱塑膠線狀材料並一層一層堆疊，來構築立體物件。這種技術操作簡單、成本較低，適合原型製作與日常應用。不過，由於成型是用層層擠出方式，成品表面通常會有層紋，精細度相對有限。

- 👉 好似 **熱熔膠槍 + 自動畫圖機**
- 👉 噴嘴加熱 → 膠絲熔化 → 一層層疊起 → 變成立體物件

**常見物料**
- PLA（最易用、最平）
- PETG（較耐熱）
- TPU（軟膠）

**FDM 點樣運作？**

1. 放入 **膠絲（Filament）**
2. 噴嘴加熱（約 180–260°C）
3. 機器跟住模型路徑「畫」
4. 一層乾咗再疊下一層

📏 每層厚度通常：

- 0.1mm（較細緻）
- 0.2mm（最常用）
- 0.3mm（快，但粗）

**常見 FDM 材料**:
| 材料 | 特點 | 家用 |
| --- | --- | --- |
| **PLA** | 易打、無乜味 | ⭐⭐⭐⭐⭐ |
| **PETG** | 耐熱、防水 | ⭐⭐⭐⭐ |
| **ABS** | 硬、耐撞 | ⭐⭐（有味） |
| **TPU** | 軟膠、彈性 | ⭐⭐⭐ |

**優點**
- 價錢低
- 操作簡單
- 無毒、氣味低
- 適合家用
- 適合原型製作

**缺點**
- 表面有層紋
- 細節有限

**適合打印**
- 手機殼
- 工具、支架
- 大型模型
- 不追求極細節的擺設

## Resin樹脂打印(光固化（LCD / DLP / SLA）)
**原理**：\
用 UV 光逐層固化液態樹脂, 光固化是一種透過光源照射液態光敏樹脂，使其逐層硬化成型的高精度列印技術。透過特定波段的光源照射光敏樹脂，讓材料逐層硬化，構築出完整模型。 相較於 FDM，光固化列印出來的模型表面更滑順、細節表現更出色，常見於公仔原型、精密零件與牙科模型等應用。 

你可以把光固化想像成「用光在液體中畫圖」。設備會選擇性地曝光樹脂槽中的特定區域，讓這些區域的液態材料硬化成形。每完成一層固化，平台就往上移動一點，再曝光下一層。

光固化之所以比 FDM 更精緻，是因為它透過「像素級的光線」準確控制每一個固化位置。相較之下，FDM 是透過擠出加熱後的材料來堆疊，而這種材料在流動與冷卻的過程中較難維持極高的精度。簡單來說，光的控制範圍比熱熔材料更小、更穩定，因此能呈現像人臉、雕花、布紋這類細緻且曲面的結構，成品表面也更加平滑自然。

**常見物料**
- Standard Resin
- ABS-like Resin
- Water-washable Resin（新手友好）

**優點**
- 細節極高
- 表面非常平滑
- 最接近商業 figure 質感

**缺點**
- 要清洗＋UV 固化
- 有氣味，要通風
- 樹脂成本較高
- 列印後需清洗與二次固化處理
- 大多數設備只能單色列印
- 設備列印尺寸通常較FDM小
- 樹脂氣味較明顯，操作環境建議通風

**適合打印**
- Figure
- 食物模型（蛋糕、壽司、漢堡）
- 小型高精度模型


### FDM vs Resin 

| 項目 | FDM | Resin |
|----|----|----|
| 細節 | ⭐⭐ | ⭐⭐⭐⭐ |
| 表面平滑 | ⭐⭐ | ⭐⭐⭐⭐ |
| 新手友好 | ⭐⭐⭐⭐ | ⭐⭐ |
| 家用安全 | ⭐⭐⭐⭐ | ⭐⭐ |
| 成本 | 低 | 中 |
| Figure 效果 | 一般 | 非常好 |
| 食物模型 | 一般 | 最佳 |
